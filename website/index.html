Install the Claude Code skill:

          
mkdir -p ~/.claude/skills/onboard-computer && curl -sL https://onboard.computer/skill.md -o ~/.claude/skills/onboard-computer/SKILL.md

          
          
Then in any project, run /onboard-computer to generate a .onboard config from your repo.
